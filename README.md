# Phone OTP Verification App

A simple React app for phone number OTP verification using Firebase.

![App Screenshot 1](https://user-images.githubusercontent.com/75136330/212552633-fe138d61-89da-4ece-9497-a6c779e50f33.png)

![App Screenshot 2](https://user-images.githubusercontent.com/75136330/212552636-ad154d82-2358-4de7-b6b5-c6e68c8b918e.png)

## Features
- Phone number input with country code selection.
- OTP generation and verification.
- Firebase authentication with reCAPTCHA.
